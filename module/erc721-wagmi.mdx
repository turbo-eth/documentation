The module exports core ERC721 hooks and components for a React application. 

[WAGMI CLI](https://wagmi.sh/cli/getting-started) (`@wagmi/cli`) is used to auto-generate hooks and other frontend application primitives. Minimal ERC721 components are also included. Components have no initial styling or design.
